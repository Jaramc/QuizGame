## 🔥 Configuración Inicial

### Orden Recomendado

```
1. Leer README principal (../README.md)
   ↓
2. Instalar dependencias (npm install)
   ↓
3. Seguir FIREBASE_SETUP.md
   ↓
4. Configurar credenciales (config/firebase.ts)
   ↓
5. Ejecutar app (npx expo run:android)
```
